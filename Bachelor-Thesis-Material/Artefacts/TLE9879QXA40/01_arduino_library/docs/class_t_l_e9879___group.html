TLE9879_Group::TLE9879_Group ( uint8_t  boardCount)

Creates a Shield group object with the specified number of Shields in the stack.

Parameters
[in]boardCountnumber of boards in the stack

Initiliazes Serial and SPI connection of the Arduino. The Serial connection has a 9600 BAUD rate. All Shields are reset before they are initialized. The constructor will try to connect to as many Shields as specified by the user. If a shield is not available and the connection fails, the constructor tries again, until the Shield answers with a valid response. This can cause an infinit loop, if one of the Shields is not properly attached or does not work properly.

Changes the specified parameter to the given value.

Parameters
[in]parameterthe parameter to change
[in]valuethe new value for the parameter
[in]boardnroptional parameter to only send this command to a specific Shield

The value of the parameter is checked before sending, in case they are not invalid. Parameters cannot be changed while the motor is running. The motor is automatically turned off, when this method is called. This method does not change the parameter in the dataset on the Arduino. To update the dataset on the Arduino, it has to be read again.
